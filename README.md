
Pet Adoption Management System

A web-based Pet Adoption Management System developed using PHP, MySQL, HTML, CSS, JavaScript, Bootstrap 5, and Dompdf. The system helps streamline the pet adoption process by allowing users to browse available pets, submit adoption applications, and generate professional PDF adoption forms. Administrators can efficiently manage pets, users, and adoption records through an admin dashboard.

## Features
Users:
- User Registration & Login
- Browse Available Pets
- View Pet Details
- Submit Adoption Application
- Generate Adoption Form as PDF
- View Adoption Status

Administrator
- Secure Admin Login
- Manage Pet Records (Add, Edit, Delete)
- Manage User Accounts
- Review Adoption Applications
- Approve or Reject Applications
- Generate Reports
